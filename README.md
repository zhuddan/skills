# skills

个人 skill 集合，遵循 [agentskills.io](https://agentskills.io) 标准。

## 安装

```bash
# 安装所有 skill
npx skills add zhuddan/skills

# 只安装某一个
npx skills add zhuddan/skills --skill hello

# 全局安装
npx skills add zhuddan/skills -g
```

## Skills 列表

| skill | 说明 |
|-------|------|
| test  | 测试 skill 是否加载成功 |
| hello | 打招呼，返回"你好" |
